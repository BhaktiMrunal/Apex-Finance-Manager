# Project Charter: Apex Finance Manager

## Objective
To develop a high-performance, full-stack Personal Finance Management (PFM) tool that enables users to track transactions and manage budgets with real-time analytics.

## Tech Stack
- **Backend:** .NET 9 Web API (C#)
- **Frontend:** Angular 19 (Signals & Deferrable Views)
- **Database:** MS SQL Server
- **DevOps:** Azure CI/CD Pipelines
